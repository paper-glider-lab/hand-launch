# フリーフライト機のハンドランチってなに？
操縦装置など必要ないので，手軽に楽しめる模型グライダーです．
紙でつくる「紙飛行機」がもっとも手軽ですが，バルサ材やカーボンなどを使った大型機もあります．
紙飛行機＝おりがみ飛行機をイメージするかもしれませんが，ここで言う「紙飛行機」はケント紙（厚紙）などで作る
紙グライダーのことです．バランスなどを調整が成功すれば３０秒以上の飛行も可能です．

紙グライダーの大きさは大きくても３０〜４０cm程度ですが，バルサグライダーは100cmを超えることもあり，
飛ばす場所がかなり広い必要があります．

詳しくは[wikiページ](https://github.com/paper-glider-lab/hand-launch/wiki)
