# フリーフライト機ってなに？
操縦装置など必要ないので，手軽に楽しめる模型グライダーのことをフリーフライトグライダー(FF glider)と呼びます．
紙でつくる「紙飛行機」がもっとも手軽ですが，バルサ材やカーボンなどを使った大型機もあります．
紙飛行機＝おりがみ飛行機をイメージするかもしれませんが，ここで言う「紙飛行機」はケント紙（厚紙）などで作る
紙グライダーのことです．バランスなどを調整が成功すれば３０秒以上の飛行も可能です．
このプロジェクトでは，その機体の構成やバランス調整方法などの話題が中心になると思います．

紙グライダーの大きさは大きくても３０〜４０cm程度ですが，バルサグライダーは100cmを超えることもあり，
飛ばす場所がかなり広い必要があります．

管理人は紙グライダー（飛行機）を主に飛ばすので，その内容が多くなります．

# ハンドランチってなに？
グライダーを上空に上げるには，ゴム動力プロペラ，カタパルトランチ，そしてハンドランチなどの種類があります．
ゴム動力で推進する竹ひごでつくった模型飛行機は以前はよくありましたが，２１世紀に入ってあまり見なくなりました．
このプロジェクトでは主にハンドランチ，つまり手投げでランチ（発航）するグライダーが中心になると思います．
